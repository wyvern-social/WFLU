# Talyx

Welcome to **Talyx**, the lightweight i18n library for managing localized messages, pluralization, and dynamic variables in your projects.

> **Quick Note:**  
> If this is **not** the `Talyx` repository and you want to list your own tool, note that this file is synced across all official Talyx docs automatically.  
> Please add your tool directly in the [`/docs/index.md` section of the Talyx repository](https://github.com/wyvern-social/Talyx/blob/main/docs/index.md#community).

## Features

- Lightweight, fast, and easy to use
- Full support for pluralization and choice messages
- Load multiple files per language; they merge automatically
- Fully community-extensible

Check out the [Quick Start](https://talyx.wyvern.gg/user-guide/quick-start/) guide for usage examples and file format details.
